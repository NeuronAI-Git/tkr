Usage
=====

.. _installation:

Installation
------------

To use tkr, first install it using pip:

.. code-block:: console

   (.venv) $ pip install tkr

-

For example:

>>> import tkr
>>> tkr.Tool(method, name, version, description, id, type)
[tkr.Tool.method, tkr.Tool.name, tkr.Tool.version, tkr.Tool.description, tkr.Tool.id, tkr.Tool.type]
